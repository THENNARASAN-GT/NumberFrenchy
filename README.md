# NumberFrenchy

## Description
NumberFrenchy is a Java console application that checks different mathematical number properties.

The program can identify:
- Spy Number
- Neon Number
- Automorphic Number

## Features
- Takes an integer as input from the user.
- Checks whether the number is:
  - Spy Number
  - Neon Number
  - Automorphic Number
- Displays the result for each check.

## Technologies Used
- Java
- Scanner Class
- Loops (while)
- Conditional Statements (if-else)

## How to Run
1. Compile the program:
   javac NumberFrenchy.java

2. Run the program:
   java NumberFrenchy

3. Enter an integer when prompted.

## Example

Input:
```
Enter an integer to check:
9
```

Output:
```
The number is not an Automorphic Number.
The number is not a Spy Number.
The number is a Neon Number.
```

## Concepts Used
- Number Manipulation
- Modulus Operator (%)
- Integer Division (/)
- Looping
- Mathematical Logic
